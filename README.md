# 🚲 서울시 공공자전거 '따릉이' 이용 및 고장 내역 분석

서울시 공공자전거(따릉이) 데이터를 활용하여 자치구별 이용 패턴을 분석하고, 고장 신고 현황을 파악한 데이터 분석 프로젝트입니다.

## 1. 프로젝트 목적
- 자치구별 대여 및 반납 현황 파악을 통한 수요 중심지 확인
- 이용량이 집중되는 특정 대여소(Hot-spot) 추출
- 자전거 고장 내역 데이터와 결합한 운영 효율화 인사이트 도출

## 2. 사용 데이터셋
- **따릉이 이용 데이터**: 자치구별 대여소 위치, 거치대 수, 대여/반납 건수 정보
- **따릉이 고장신고 데이터**: 자전거 번호별 고장 등록 일시 및 부위 정보

## 3. 주요 분석 내용 및 결과
### 📍 자치구별 이용 현황
- **강서구**가 서울시 전체 자치구 중 대여량 1위를 차지함 (약 220만 건).
- 송파구와 영등포구가 그 뒤를 이어 높은 이용량을 보임.

### 📍 주요 대여소 분석
- **마곡나루역 2번 출구** 대여소가 가장 높은 이용 빈도를 기록함.
- 주요 지하철역 인근 대여소에 이용객이 집중되는 경향을 확인.

### 📍 데이터 품질 확인
- 위도/경도 및 일부 거치대 수 데이터에서 결측치가 발견되어 전처리 수행.
